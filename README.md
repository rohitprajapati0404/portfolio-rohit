# Professional Portfolio Website

A production-ready, highly interactive static portfolio website built with React and Framer Motion.

## 🚀 Features

- **Dynamic UI**: Modern, minimalist design with a premium feel.
- **Interactive Animations**: Smooth transitions using Framer Motion and AOS (Animate on Scroll).
- **Responsive Design**: Mobile-first approach, fully optimized for all devices.
- **Experience Timeline**: Interactive journey showcase with expandable cards.
- **Interactive Form**: Built-in validation and simulated submission logic for high engagement.
- **Skill Progress**: Animated skill bars and interactive technology icons.

## 🛠️ Technology Stack

- **Frontend**: React 19, CSS3 (Flexbox/Grid), Framer Motion, AOS, React Icons.
- **Build Tool**: Vite.
- **Styling**: Vanilla CSS with modern variables and glassmorphism.

## ⚙️ Setup & Installation

### 1. Installation
```bash
cd frontend
npm install
```

### 2. Development
```bash
npm run dev
```
The application will run on `http://localhost:5173`.

### 3. Build for Production
```bash
npm run build
```

## 📝 Usage

- Edit `frontend/src/sections/Hero.jsx`, `About.jsx`, etc., to customize your personal information.
- Replace the resume file in `frontend/public/resume.pdf`.
